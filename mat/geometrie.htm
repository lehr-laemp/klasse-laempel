Klasse Lämpel: Geometrie

Grundkonstruktionen

Punkte zeichnen und anschreiben

Punkte werden mit einem kleinen Kreuz gezeichnet.
Sie werden mit grossen Buchstaben angeschrieben.

Geraden zeichnen und anschreiben

Geraden werden mit kleinen Buchstaben angeschrieben.
Sie haben keinen Anfang und kein Ende.

Strecken zeichnen und anschreiben

Strecken werden mit kleinen Buchstaben angeschrieben.
Sie haben auf beiden Seiten einen Punkt.
Man kann auch schreiben Strecke AB oder BC.

Strecken messen mit dem Geodreieck

Das Geodreieck so hinlegen, dass die Null beim ersten Punkt ist.
Dann kann man die Länge beim zweiten Punkt ablesen. Beispiel bei Youtube

Parallelen zeichnen mit dem Geodreieck

Man legt das Geodreieck an die Gerade.
Man verschiebt das Geodreieck und schaut auf die Striche. Beispiel bei Youtube

Senkrechten zeichnen mit dem Geodreieck

Lege die Mittellinie des Geodreiecks auf die Gerade.
Zeichne entlang der Messleiste eine Gerade. Beispiel bei Youtube

Winkel messen mit dem Geodreieck

Man legt das Geodreieck so in den Winkel:
Die 0 muss auf der Spitze des Winkels liegen. Beispiel bei Youtube
